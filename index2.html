⚠️ Setup Required for Family Sharing

To enable real-time syncing between family members, you need to set up Firebase (free):

  1. Go to Firebase Console
  2. Create a new project (free)
  3. Go to Project Settings → General
  4. Scroll down and click "Add app" → Web app
  5. Copy your Firebase config
  6. Replace the placeholder config in this HTML file (around line 700)
  7. Go to Firestore Database → Create Database → Start in test mode
  8. The tracker will sync automatically once configured!
